# Добавляем в очередь задачи по отправке сообщений

## Что имеем:

* Ruby 3.0.3
* Rail 7
* Esbuild для сборки assets
* Bootstrap - стили
* PostgreSQL - БД

## Для запуска
Выполняем команды: bundle, rails css:install:bootstrap, yarn build:css, yarn build, rails db:create, rails db:migrate;
